== Description ==

Simple FAQ contents management tool.

== Installation ==

1. Upload the plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create FAQ content from the FAQ menu
4. Place `[bfaq_faq_tabs]` shortcode in your contents or `<?php do_shortcode( 'bfaq_faq_tabs' ); ?>` in your templates
